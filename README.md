# R3BBIT 🐰
A high-performance, minimalist parenting utility built for speed, data sovereignty, and infant soothing.

## ✨ Features
*   **The Soothe Engine**: Low-latency white noise loops using `SoundPool` with a safety fade-out algorithm to prevent sudden silence from waking the baby.
*   **R3 Scheduling**: Dynamic Wake Window recalibration that adjusts your next "Ideal Nap Time" based on the quality of the previous sleep.
*   **The Toolbox**: Full data sovereignty. Import, export, and merge logs via CSV or simple clipboard sync.
*   **Accessibility First**: Includes a **Left-Handed Mode** for easier one-handed operation while holding a child.
*   **Visual Trends**: Custom-drawn canvas charts for feeding volumes and sleep patterns.

## 🛠️ Tech Stack
*   **Language**: Kotlin
*   **Architecture**: Foreground Services for background audio
*   **Storage**: Offline-first SharedPreferences with CSV logic
*   **UI**: Custom Canvas drawing and dynamic FrameLayouts

## 🚀 Development
To test the "Pro" features without a billing account, navigate to **Settings** and tap the **Version String** at the bottom 5 times to toggle the debug unlock.
